# Web Project 01

これは Spring Boot をベースにした Web 管理システムの学習プロジェクトです。情報の登録・操作履歴の記録・統計など、実務に近い機能を段階的に実装しています。

## 📁 プロジェクト構成

## 🚀 使用技術

- Spring Boot
- Spring AOP
- MyBatis
- Lombok
- MySQL（環境に応じて切替）
- Maven

## 🧩 実装済み機能

- 情報の登録・更新・削除・検索
- 操作履歴の自動記録（AOPによる切り出し）
- 操作統計の集計表示
- ファイルアップロード（社員画像など）
- ログ出力・例外処理の共通化
